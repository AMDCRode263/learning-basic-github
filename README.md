# Learning Basic GitHub Repository

This repository is created as a place to learn and share code related to Internet of Things (IoT), robotics, and machine learning.  
The main goals of this repository are to:

- Help myself and others understand the basics of using GitHub and Git.  
- Store simple example codes that can serve as references for IoT and robotics projects.  
- Facilitate collaboration and joint development with the open-source community.

All code here is open source and free to study, use, and further develop, with attribution to the original creator.

---

### How to use this repository:

1. Study the example codes provided.  
2. Upload and share your own code for further development.  
3. Use Issues and Pull Requests features to discuss and collaborate.

---

Hopefully, this repository will be useful for all of us who want to learn and grow in the fields of IoT and programming!
